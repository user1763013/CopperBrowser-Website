🚀 CopperBrowser V1

A lightweight, open-source browser built with Python 3.13, PyQt6, and QtWebEngine.

Fast. Simple. Extendable.

Tabs, bookmarks, history, and more — all in a minimal package.

  • Bookmarks & history saved locally
  • User agent & search engine switching
  • Lightweight and open-source
