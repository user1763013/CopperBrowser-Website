Features

  • Tabs: Multiple independent browsing sessions.
  • Toolbar: Back, forward, reload, home.
  • Bookmarks: Stored in %APPDATA%\CopperBrowserV1\bookmarks.json.
  • History: Stored in %APPDATA%\CopperBrowserV1\history.json.
  • User agent: Toggle Copper vs Chrome.
  • Search engine: DuckDuckGo, Google, and more.
  • Config: %APPDATA%\CopperBrowserV1\config.json.
